# Frontend (LearnTube)

React + Vite frontend for the LearnTube project.

For full project documentation, architecture, deployment, and roadmap, see:
- `../README.md`

## Commands

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`

## Environment variable

Set this in local `.env` or deployment platform:

- `VITE_API_BASE_URL=https://<your-backend-url>`

If not set, code falls back to localhost backend.
